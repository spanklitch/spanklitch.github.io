Claude Chat Desktop Shortcut

Creates a desktop shortcut for Claude AI Chat on Linux (XFCE/Xubuntu) with the Claude logo icon.

One-liner install:

curl -s https://raw.githubusercontent.com/spanklitch/spanklitch.github.io/main/claude-chat-shortcut.sh | bash

What it does:

  • Downloads the Claude AI logo
  • Creates a clickable desktop shortcut
  • Opens Claude Chat in Google Chrome
